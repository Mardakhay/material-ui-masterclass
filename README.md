# Material UI Interactive Masterclass

An interactive, MUI-inspired learning page that teaches Material UI setup, theming, core components, and a complete login form example with live demos.

## What it includes

- Vite + React + TypeScript setup guide
- MUI installation and Roboto font setup
- ThemeProvider and createTheme explanation
- Button, TextField, and Card component demos
- Interactive login form example

## How to use

Open the HTML file in a browser and use the lesson navigation to explore each section.

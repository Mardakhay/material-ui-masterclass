What is Material UI?

Material UI (MUI) is a widely used React UI component library built on Google's Material Design system.

🎨 50+ Components ⚡ Zero CSS needed ♿ Built-in a11y 📱 Responsive 🔧 Fully customizable
The Core Concept
Instead of writing CSS from scratch, you use pre-built components that follow Google's Material Design specification. One line of code gives you a fully animated, accessible, production-ready button.
App.tsx 
import { Button } from '@mui/material';

function App() {
  return (
    <Button
      variant="contained"
      color="primary"
      size="large"
    >
      Send
    </Button>
  );
}

// Result: a beautiful, animated, accessible button 🎉
Live Result
💡
Key insight

That button has hover effects, ripple animation, focus states, keyboard accessibility, and Material Design styling — all for free with zero CSS.

Custom UI Kit vs Material UI
Understanding when to use each approach.
🎨 Custom UI Kit
✍️ Write everything by hand
🎯 100% control — it's your code
📦 Smaller bundle size
🖌️ Unique design per project
⏱️ Takes much more time
🐛 You handle every edge case
📖 Great for learning
📦 Material UI
🚀 Ready-made components
📐 Material Design standards
♿ Accessibility built-in
🌍 Huge ecosystem & community
⚡ Extremely fast development
🧪 Battle-tested, stable
💼 Used in real companies
🏢
Real-world verdict

In production, most companies use ready-made UI libraries (MUI, Ant Design, Chakra UI). A custom UI kit is mainly for learning or highly specific design requirements.

Lesson 4 of 8

ThemeProvider

The central configuration system. Wrap your entire app with ThemeProvider and every component inherits your design tokens — colors, fonts, border-radius, and more.

Full Configuration Code
src/App.tsx 
import { CssBaseline } from '@mui/material';
import { ThemeProvider, createTheme } from '@mui/material/styles';
import Typography from '@mui/material/Typography';

// 1) Create your theme
const theme = createTheme({
  typography: {
    fontFamily: "'Roboto', sans-serif",
  },
  shape: {
    borderRadius: 10,  // corner roundness in px
  },
  palette: {
    primary:   { main: '#1976d2' },  // blue
    secondary: { main: '#9c27b0' },  // purple
  },
});

function App() {
  return (
    // 2) Wrap your app with ThemeProvider
    <ThemeProvider theme={theme}>
      {/* 3) CSS reset — normalizes browser defaults */}
      <CssBaseline />

      <div style={{ padding: '2rem' }}>
        <Typography variant="h3">
          🎉 MUI Connected!
        </Typography>
      </div>
    </ThemeProvider>
  );
}
export default App;
createTheme Parameters
ParameterDescriptionExample
typography.fontFamilyFont for all text across the app'Roboto', sans-serif
shape.borderRadiusDefault border-radius in pixels10
palette.primary.mainPrimary color — used by Buttons, Inputs#1976d2
palette.secondary.mainSecondary color accent#9c27b0
How ThemeProvider Works (Diagram)
Every child component automatically receives the theme. No props needed.
<ThemeProvider theme={theme}>
├── <CssBaseline /> ← normalizes browser CSS
├── <Button /> ← gets primary color automatically
├── <TextField /> ← gets border-radius automatically
└── <Card /> ← gets typography automatically
</ThemeProvider>
✅
Test it

Run npm run dev — you should see "🎉 MUI Connected!" in the browser. That's Roboto font, MUI Typography, with your theme applied.

Full Source Code
src/App.tsx — Complete Login Form 
import { useState } from 'react';
import { CssBaseline, Button, TextField, Card, CardContent,
  CardActions, CardHeader, Divider, Stack } from '@mui/material';
import { ThemeProvider, createTheme } from '@mui/material/styles';
import SendIcon from '@mui/icons-material/Send';

const theme = createTheme({
  typography: { fontFamily: "'Roboto', sans-serif" },
  shape: { borderRadius: 10 },
  palette: {
    primary:   { main: '#1976d2' },
    secondary: { main: '#9c27b0' },
  },
});

function App() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  const emailError = email && !email.includes('@')
    ? 'Invalid email format' : '';

  return (
    <ThemeProvider theme={theme}>
      <CssBaseline />
      <div style={{
        minHeight: '100vh', display: 'flex',
        alignItems: 'center', justifyContent: 'center',
        background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)',
      }}>
        <Card elevation={8}
          sx={{ maxWidth: 420, width: '100%', borderRadius: 3 }}>
          <CardHeader
            title="🔐 Sign In"
            subheader="Sign in to your account"
          />
          <CardContent>
            <Stack spacing={2.5}>
              <TextField
                label="Email" type="email"
                value={email}
                onChange={e => setEmail(e.target.value)}
                error={!!emailError}
                helperText={emailError}
                fullWidth
              />
              <TextField
                label="Password" type="password"
                variant="filled"
                value={password}
                onChange={e => setPassword(e.target.value)}
                fullWidth
              />
            </Stack>
          </CardContent>
          <Divider />
          <CardActions sx={{ justifyContent: 'space-between', p: 2 }}>
            <Button variant="text">Forgot Password?</Button>
            <Button
              variant="contained"
              startIcon={<SendIcon />}
              disabled={!email || !password}
              onClick={() => alert('Login!')}
            >
              Sign In
            </Button>
          </CardActions>
        </Card>
      </div>
    </ThemeProvider>
  );
}

export default App;
